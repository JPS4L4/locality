Locality

Es una tienda virtual enfocada en ayudar a los pequeños comerciantes

Para instalar Locality en un servidor o conectar a una base de datos, se deberá ir a la carpeta Config y 
luego al archivo config.php una vez estando allí se podrá modificar el puerto, la url, etc. Para la correcta 
conexión a un servidor web, en este archivo también se podrá encontrar la conexión a la base de datos, 
es donde se recibirá todos los datos como productos, imagenes, precios, usuarios, entre otros. 

En caso tal de que no funcione la conexión a la base de datos o al servidor web elimina el caché del navegador y
verifica todo tipo de cambios que realices en el archivo config.php para su correcto funcionamiento.

Esto sería suficiente para instalar correctamente el aplicativo, en caso de requerir más información recuerda
leer el manual o documentación de la aplicación Locality allí encontrarás información detallada de todo el código.
